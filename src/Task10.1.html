TypeScript:

Gibt es statische Klassen in TypeScript?

Ja, es gibt in TypeScript statische Klassen.
Sie werden mit dem Schlüsselwort "static" definiert und können nicht instanziiert werden.
Mitglieder einer statischen Klasse können nur über die Klasse selbst aufgerufen werden, nicht über eine Instanz der Klasse.

Was ist der Unterschied von TS zu anderen objekt-orientierten, statisch typisierten Programmiersprachen wie Java und C#?

Der Unterschied zwischen TypeScript und anderen objekt-orientierten, statisch typisierten Programmiersprachen wie Java und C# besteht darin, dass TypeScript eine Erweiterung von JavaScript ist und daher die Flexibilität und Dynamik von JavaScript beibehält.
Diese Flexibilität äußert sich zum Beispiel in der hohen Kompatibilität oder der Prototypischen Vererbung von JavaScript.
